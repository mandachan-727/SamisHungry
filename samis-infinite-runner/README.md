# SamisHungry Infinite Runner Game

## Overview
SamisHungry is an infinite runner game featuring a character named Sam, who runs automatically from left to right. Players must help Sam jump to collect positive items while avoiding negative items. The game ends when Sam reaches exactly 138 points and clicks on a treasure chest.

## Features
- Automatic running from left to right
- Jumping mechanics to collect positive items
- Avoidance of negative items
- Game ends upon reaching 138 points and clicking a chest

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd samis-infinite-runner
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To start the game, run:
```
npm start
```

## Game Mechanics
- **Running**: Sam runs automatically.
- **Jumping**: Use the spacebar to make Sam jump.
- **Scoring**: Collect positive items to increase your score.
- **End Game**: Reach exactly 138 points and click on the chest to end the game.

## Assets
Ensure that all game assets are properly loaded in the BootScene before transitioning to the PlayScene.
